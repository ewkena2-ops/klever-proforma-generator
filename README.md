# Proforma Generator

Standalone local proforma generator.

- Open `index.html` in a browser.
- No API or dependency install is used.
- Use `Save PDF` for PDF printing or `Save Excel` for CSV export.
- On iPhone and iPad, the PDF opens in a browser tab so it can be saved or shared from the iOS share sheet.
- Use `Save Project` to download an editable JSON backup, then `Load Proforma` to restore and continue editing later.
- `Load Proforma` can also reopen CSV files created by `Save Excel`.
- Set Document Status to `Cancelled` to mark the proforma as cancelled in the preview and PDF.
- Choose whether the proforma is with or without installation, and with or without accessories; the terms update automatically.
- Terms and conditions always exclude granite, marble, quartz, and countertops.
- Item quantity multiplies area, price, total area, VAT, and final total.
- Cabinet material is split into inside material, outside finish, and quality level. Outside finishes include UV / HDF, UV / Solid Wood, Super Matt / HDF, Super Matt / Solid Wood, Laminate MDF, and Painted MDF.
- New cabinet rows start with no outside finish selected.
- Acknowledgement joins material options with `or`; Amendment joins them with `and`.
- Aluminium rows hide cabinet-only material controls.
- Item rows can be duplicated for faster repeated room/item entry.
- Excel export includes separate inside material, outside finish, quality, and final material sentence columns.
- On mobile, item rows switch to a stacked card layout.
- The logo is bundled locally as `logo.png`.

All files stay inside this folder.
